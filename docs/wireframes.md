# Wireframes low-fi

Les wireframes suivants décrivent la structure de contenu attendue. Chaque bloc correspond à une section verticale. Les éléments interactifs sont notés `[CTA]`, les éléments visuels `(image)` ou `(vidéo)`.

## 1. Accueil (`/`)
```
[HEADER STICKY]
Logo | Accueil | Offres | Méthode & ROI | Cas clients | Ressources | Financements | À propos | [CTA Diagnostic]
-----------------------------------------------------------------------------------------------------------
| HERO FULL WIDTH                                                                                         |
|  - H1 : "Coaching intensif à la vente pour PME B2B (Montpellier & Paris)"                               |
|  - Sous-titre : "3 sessions en visio + 1 journée présentielle..."                                       |
|  - [CTA primaire Diagnostic gratuit]    [CTA secondaire Télécharger le programme]                       |
|  - (visuel dynamique : séance coaching)                                                                 |
-----------------------------------------------------------------------------------------------------------
| BARRE KPI ROI                                                                                            |
|  - "95 % des équipes signent plus vite" | "+10 pts taux de transfo" | "5 000 € pour 6"                 |
-----------------------------------------------------------------------------------------------------------
| SECTION PREUVES & ROI                                                                                    |
|  - Titre H2 : "Une méthode 100 % orientée résultats"                                                   |
|  - 4 colonnes : Évaluation → Objectifs → Entraînement → Suivi                                            |
|  - Encadré pricing à droite : tarif + financement OPCO                                                   |
-----------------------------------------------------------------------------------------------------------
| SECTION CAS CLIENTS (teaser)                                                                             |
|  - H2 : "Des PME locales qui ont accéléré"                                                              |
|  - Slider 3 cartes étude de cas (image + résultats + [CTA Voir cas])                                     |
-----------------------------------------------------------------------------------------------------------
| SECTION OFFRES                                                                                            |
|  - 3 cartes horizontales : Sprint 1 mois | Parcours 3 mois | Workshop Flash                              |
|  - Chaque carte : durée, modules, bénéfices, [CTA Découvrir]                                             |
-----------------------------------------------------------------------------------------------------------
| SECTION MÉTHODE (vidéo)                                                                                  |
|  - (Vidéo témoignage/présentation Laurent) + bullet points                                               |
-----------------------------------------------------------------------------------------------------------
| SECTION TRUST                                                                                            |
|  - Logos CCI, French Tech, partenaires                                                                   |
|  - Citation client + notation étoiles                                                                    |
-----------------------------------------------------------------------------------------------------------
| SECTION CTA FINAL                                                                                        |
|  - H2 : "Prêt à doper vos ventes ?"                                                                     |
|  - [CTA Diagnostic gratuit] + lien Calendly                                                               |
-----------------------------------------------------------------------------------------------------------
| FOOTER                                                                                                   |
|  - Coordonnées, NAP, liens légaux, newsletter, réseaux sociaux                                           |
-----------------------------------------------------------------------------------------------------------
```

```
-----------------------------------------------------------------------------------------------------------
| SECTION PACKAGES                                                                                         |

-----------------------------------------------------------------------------------------------------------
```
