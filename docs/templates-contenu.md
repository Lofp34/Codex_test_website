# Templates de contenu

## 1. Template article de blog
```
Titre (H1) : [Mot-clé principal] — [Bénéfice concret]
Meta title : [60 char max]
Meta description : [155 char max]
Slug : mot-cle-vente

[Hero]
- Image 1200×630 (WebP) + alt text descriptif
- Chapô (2-3 phrases impact)
- Badge catégorie (ex : Prospection, Closing)

[Structure]
H2 #1 — Contexte / Problématique
  - Paragraphe 2-3
  - Encadré statistique (icône + KPI)
H2 #2 — Méthode ou framework
  - Liste numérotée
  - Citation Laurent Serre
H2 #3 — Plan d'action 30/60/90 jours
  - Tableau ou bullet points
  - CTA interne vers cas client

[Encarts]
- Bloc téléchargement (PDF, template) avec mini-form HubSpot
- Bloc "Envie d'aller plus vite ?" + CTA Diagnostic

[Conclusion]
- Résumé 3 points clés
- CTA principal Diagnostic
- CTA secondaire Inscription webinar

[SEO]
- 2 liens internes (offres, cas)
- 1 lien externe autorité
- Balise FAQ optionnelle (3 Q/R)
```

## 2. Template étude de cas
```
Titre (H1) : Comment [Entreprise] a [résultat] en [délai]
Meta title : Cas client — [Entreprise] (Secteur)
Meta description : Résumé du challenge et des gains chiffrés.
Slug : cas-[entreprise]-[ville]

Hero
- Logo client + photo équipe
- Résultats clés (3 KPI) + contexte (secteur, taille équipe)
- CTA Télécharger le cas PDF

Section 1 — Le challenge
- H2 + paragraphe
- Liste freins (avant le coaching)

Section 2 — L'approche Laurent Serre
- Timeline 4 étapes (Diagnostic, Sprint, Coaching terrain, Suivi)
- Encadré outils (scripts, jeux de rôle filmés)

Section 3 — Résultats
- Graphique (bar chart) montrant progression KPI
- Témoignage vidéo + transcription

Section 4 — Facteurs clés de succès
- 3 bullet points
- Citation Laurent

Section 5 — Et la suite ?
- Mentoring manager, plan d'action 90 jours

CTA final — Diagnostic gratuit (avec ancre formulaire)

SEO
- Donnée structurée `Service` + `Review` (si note)
- Liens internes vers pages offres/locales
```

## 3. Template landing webinar
```
Titre (H1) : Webinar — [Sujet orienté bénéfice]
Meta title : Webinar vente B2B — [Sujet]
Meta description : Date + bénéfices + bonus.
Slug : webinar-[sujet]

Hero
- Date + heure + durée (icônes)
- Compte à rebours (component)
- CTA Inscription (HubSpot form inline)

Section 1 — Ce que vous apprendrez
- Liste 3-4 points, orientés ROI
- Encadré bonus (template offert)

Section 2 — Intervenants
- Photo Laurent + bio courte
- Co-intervenant (si applicable)

Section 3 — Pour qui ?
- Personas + critères (CA, taille équipe)

Section 4 — Agenda détaillé
- Timeline 4 segments (Prospection, Discovery, Négociation, Q&A)

Section 5 — Cas client teaser
- Mini témoignage vidéo ou stat

Section 6 — Après le webinar
- Accès replay 48h, PDF slides
- CTA secondaire "Recevoir le programme" (form HubSpot minimal)

FAQ (4 questions)
- Rediffusion ?
- Requis techniques ?
- Limite participants ?
- Certificat ?

Footer spécial
- Icônes réseau + partage LinkedIn/Twitter

Tracking
- Événements GA4 : inscription, clic CTA, téléchargement bonus
```
